# 🎧 CODING-SONG

A coding music player built from real experience, because every coding session has a different mood.  
Add your songs and code with better vibes!

---

##  Why I Built This

We don’t feel the same every day.

Sometimes we feel motivated.  
Sometimes tired.  
Sometimes even a bit disappointed.

Today, after finishing my LeetCode assessment for a job application, I had mixed feelings.  
I passed the first stage (JavaScript), which felt great.

But in the second stage, I chose JavaScript (Node.js)... honestly, I don’t even know why 😅  
Then suddenly it switched to another language (Python 2), and I couldn’t pass both parts.

It wasn’t the result I wanted.

So instead of overthinking it, I decided to build something.

I thought:  
*What if I had an app that helps me feel better while coding?*  
*Something simple… music, focus, and good vibes.*

And that’s how **CODING SONG 🎧** was created.

---

## 🎧 About This Project

This is a simple music player for developers.

It’s not just about playing songs — it’s about:
- staying focused  
- improving your mood  
- enjoying the process of coding  

---

##  Contribute

I would love other developers to contribute!

You can:
- Add your favorite coding music  
-  Improve UI/UX  
-  Add features (progress bar, themes, animations)  

---

##  How to Contribute

1. Fork this repository  
2. Add your song in `script.js`  

Example:

```js
{
  id: 4,
  title: "Your Song",
  artist: "Artist",
  src: "your-link.mp3"
}
